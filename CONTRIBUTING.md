# Contributing

Mosaic is MIT-licensed. Fork it, modify it, build on it — no permission needed.

Mosaic is solo-maintained on part-time hours. **Pull requests are unlikely to be reviewed or merged.** If you want changes, fork.

Bug reports are still welcome — file one against [mosaic_python](https://github.com/matt-mohn/mosaic_python/issues). Even if a fix doesn't ship, the report is useful as a reference for the next person who hits the same problem.
